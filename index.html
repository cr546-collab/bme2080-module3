About This Module

What You'll Explore

This module introduces machine learning approaches applied to real multi-omics cancer datasets. You'll work with RNA expression, protein expression and drug response data across multiple cancer types.

🧬

Multi-Omics Data

Integrate RNA expression and protein levels to understand how molecular features relate to cancer biology.

💊

Drug Response

Explore how genomic features predict response to 15 targeted cancer drugs, including EGFR, BRAF, PI3K, and JAK inhibitors.

📊

Interactive Visualization

Use the Multi-Omics Explorer to visualize relationships between variables, filter by cancer type, and generate scatter plots, heatmaps, and more.

🤖

ML Applications

Apply classification and regression models to predict drug sensitivity from molecular features.

How To Use

Getting Started

Follow these steps to explore the Multi-Omics Explorer.

01
Choose a Plot Type
Use the tabs on the left sidebar to switch between Scatter, Box, Heatmap, and Bar plot types.
02
Select Your Variables
Pick a data type (RNA Expression, Protein Expression, or Drug Response) and a specific gene or drug for each axis. Try exp_EGFR vs drug_Gefitinib as a starting point.
03
Filter by Cancer Type
Toggle cancer type buttons at the bottom of the sidebar to compare specific subsets. Click a button to include or exclude that cancer type from the plot.
04
Hover for Details
Hover over any data point to see the cell line ID, cancer type, and exact values for both axes.
